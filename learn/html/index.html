HTML

Extension: .html

Syntax: <tagname attribute="value">Content</tagname>

Case Sensitive: No

Usability: Easy

Required Setup:
<!DOCTYPE html>
<html>
<head>
<title> TITLE </title>
</head>
<body>
<!--Content goes here-->
</body>
</html>

Compatible languages: CSS(.css), JavaScript(.js), PHP(.php)

About HTML

HTML is one of the most easiest languages to learn coming only behind Markup(.md). HTML is one of the languages that makes up the client side of the website.
