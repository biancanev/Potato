HTML Images

Tag(s): <img>

Attribute(s): src[required], class, id, onclick, style

Example Code:
    Input:

  • <img src="http://thepotatoplace.ga/39782933-hexagon-wallpapers.jpg ">
Output:

Examples Explained

The <img> tag tells the computer that you want to place an image. Image size is original size unless otherwise specified. Use the src attribute to specify the picture url.(Note: If you uploaded a picture to your host, use your URL plus any directories you have e.g. http://thepotatoplace.ga/dir1/dir2/img.png)
